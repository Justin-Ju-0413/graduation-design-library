# Tuesday Report Workspace

This folder is the working area for the Tuesday noon progress report.

## Goal

Use the existing weekly-report PPT template and fill it with the prepared
English content plus the experiment evidence you will collect tonight.

## Files In This Folder

- `SLIDES_EN.md`
  - ready-to-paste English slide content for the main 8-page deck
- `BACKUP_QA_EN.md`
  - backup answers for likely advisor questions
- `KEY_RESULTS_PAGE_TEMPLATE.md`
  - how to build the key-results page from your screenshots and logs
- `evidence/`
  - put screenshots, logs, photos, and waveform captures here

## How To Use

1. Open your old weekly-report PPT and keep the template only.
2. Copy the slide text from `SLIDES_EN.md` page by page.
3. Put tonight's screenshots and logs into `evidence/`.
4. After you finish collecting evidence, tell me which files you added.
5. I will then help you choose the strongest figures and refine the key-results page.

## Minimum Evidence Needed Tonight

- one screenshot for local RTL simulation pass
- one screenshot for full-SoC regression pass
- one screenshot showing `expected_rstat = 19`
- one screenshot for bitstream rebuild success or Vivado success
- one screenshot for FPGA download success

## Best Extra Evidence

- UART output screenshot
- ILA waveform screenshot
- board LED observation photo
