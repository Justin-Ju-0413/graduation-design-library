# 07_Backups

This directory stores older working copies, exported snapshots, and archived drafts that are no longer actively used but are preserved for reference.

## Contents

- `thesis_drafts/` -- Previous versions of the thesis document (Draft, Formatted, etc.)

## Policy

- Only files that are superseded by newer versions should be placed here.
- This is NOT a substitute for version control. Use `git` for code and important document versioning.
- Clear old backups periodically to avoid bloat.
