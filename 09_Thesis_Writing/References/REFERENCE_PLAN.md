# Reference Plan

## Reference Groups

1. RISC-V and custom instruction foundation
   - RISC-V Unprivileged ISA Specification
   - RISC-V Privileged Architecture Specification if exception/debug context is needed
   - Nuclei E203/Hummingbird documentation
   - Nuclei NICE documentation

2. CNN accelerator background
   - DianNao
   - Eyeriss
   - Gemmini
   - other lightweight CNN accelerator papers

3. FPGA-based CNN acceleration
   - FPGA CNN accelerator surveys
   - FPGA implementation papers for convolution, PE array, and dataflow

4. RISC-V accelerator integration
   - RISC-V custom instruction accelerator papers
   - RISC-V coprocessor/CNN accelerator papers
   - RISC-V SoC verification references

5. Engineering tools and debug flow
   - Nuclei SDK documentation
   - OpenOCD documentation
   - GDB remote debugging documentation
   - Xilinx Vivado documentation where board bring-up is discussed

## Minimum Target

- 20 to 30 total references.
- 12 to 18 academic papers.
- 6 to 10 official manuals, specifications, or tool documents.

## How to Use References in the Thesis

- Chapter 1: cite CNN acceleration need and RISC-V openness.
- Chapter 2: cite accelerator architectures and RISC-V integration approaches.
- Chapter 3: cite E203/NICE documents.
- Chapter 4: cite hardware verification and CNN accelerator design papers.
- Chapter 5: cite SDK/SoC integration and simulation references.
- Chapter 6: cite Vivado, OpenOCD, GDB, and board debug documentation.
